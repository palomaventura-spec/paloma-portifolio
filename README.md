# Paloma Aguiar Portfolio

## Rodar no VS Code

```bash
npm install
npm run dev
```

## Subir no GitHub

```bash
git init
git add .
git commit -m "portfolio landing page"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/paloma-portfolio.git
git push -u origin main
```

## Deploy na Vercel

1. Acesse vercel.com
2. Add New Project
3. Importe o repositório do GitHub
4. Framework: Vite
5. Deploy
"# paloma-portifolio"
